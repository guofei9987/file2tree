# file2tree
file2tree

### file2tree.py
file2tree generate a .txt file like this：

    ├── conf
    └── test
        └── app
            ├── conf
            ├── html
            │   ├── a.js
            │   ├── bigscreen.js
            │   └── index.html
            ├── module
            │   ├── a.html
            │   └── media.html
            └── styl


### usage
in CMD：
- use like "python file2tree.py -r d:/tree"
- or use like "python file2tree.py --rootdir d:/tree"
